**Automação de Testes com Playwright – SauceDemo**

**Descrição**

Este projeto tem como objetivo demonstrar a automação de testes end-to-end utilizando o framework Playwright no site de demonstração SauceDemo (https://www.saucedemo.com/).
Foram desenvolvidos dois scripts de automação que simulam o fluxo de compra de produtos até a etapa do carrinho de compras, permitindo validar funcionalidades básicas da aplicação e praticar diferentes abordagens de criação de testes.

**Cenários Automatizados**
1. compra_produto_gen.spec.js
Script criado utilizando o recurso Playwright CodeGen, que gera automaticamente o código das ações realizadas pelo usuário durante a navegação.
Link do video de excução:
https://x.gd/LXZn5

3. compra_TShirt.spec.js
Script desenvolvido manualmente, sem utilização do CodeGen, aplicando boas práticas de automação e seletores apropriados do Playwright.
Link do video de excução:
